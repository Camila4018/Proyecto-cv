Bienvenidos al mundo de la programación

Front-End

Es un anglicismo que al español se traduce como interfaz, front-end tiene origen en las palabras en ingles "FRONT" que significa frente y "END" final.

Concepto de Frontend

Es un termino que se utiliza para definir la interfaz gráfica de un sitio online, es todo lo que el usuario puede ver. Colores, tipografías,tamaños, distribución de elementos, contenidos. Por lo tanto puede ser utilizado con frecuencia en el diseño web. También es importante definirlo como una parte del software que es capaz de interactuar con los usuarios. Los principales lenguajes de programación que utiliza son HTML,CSS y JAVASCRIPT.

Estructura de un proyecto web

  • HTML es el primer lenguaje que se debe aprender para el desarrollo web.
  • Con HTML se puede crear páginas web y solo es necesario un editor de texto.
  • Los documentos HTML contienen ELEMENTOS que se marcan con ETIQUETAS .
  • Las etiquetas son solo de APERTURA o de APERTURA Y CIERRE.
  • Los elementos pueden tener ATRIBUTOS que agregan información adicional.

Insertar videos en HTML

Para insertar videos en HTML existen dos tipos de codigos IFRAME que permite incrustar en nuestra pagina el enlace de otra pagina HTML, como por ejemplo: YouTube, para proceder con esta acción es necesario ingresar a YouTube, clickear el boton derecho de nuestro mause sobre el video que vamos a usar y seleccionar la opción copiar codigó de inserción; el segundo codigó es VIDEO el cual puede ir acompañado de los siguientes atributos:src para marcar la ruta y source para agregar distintos formatos de reproducción al video insertado ya sea mp4,ogg o webm, los cuales deben ser precedidos por el termino type; además existen otros atributos como width que nos indicara el ancho y heigth que indicara el alto de la imagen del video, controls que permitira al usuario controlar la reproducción y demás aspectos del video, autoplay autoreproducción del video loop una vez que termina el video vuelve a empezar automaticamente mute permite silenciar el audio del video elegido
